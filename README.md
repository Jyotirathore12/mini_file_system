# Mini File System (C++)

# Overview:
This project is a simplified in-memory file system implemented in C++.  
It simulates basic file system operations such as creating directories, creating files, navigating paths, and displaying directory structure.

The goal of this project is to understand how file systems work internally using core data structures.


# Features
- Create directories and files
- Maintain hierarchical directory structure
- Traverse directories using paths
- Display file system tree


# Tech Stack & Concepts Used
- Language: C++
- Data Structures:
  - Trees
  - Hash Maps
- OOP Concepts:
  - Classes & Objects
  - Encapsulation
- Memory management
- Command-based input handling
